3D Cube

This experiment renders 6 videos simultaneously in a 3D rotating cube.

  • License: Enterprise
  • Player Version: 6.11.4923
  • Github

Each face of this cube is a fully functional JW Player. This demo uses CSS 3D Transforms for the Cube’s animations, and leverages the JW Player JavaScript API for the Cube’s interaction.

A special thank you goes out to Wilfred (Willie) Meier, for the inspiration for this experiment, as he created the original 3D Cube using JW Player 5.
